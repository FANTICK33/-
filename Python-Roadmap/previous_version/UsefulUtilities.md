## Работа с EXCEL файлами
### Библиотеки:
Для чтения и записи XLS файлов, вам нужно будет установить два модуля — ```xlrd``` и ```xlwt```. Вы можете установить их с помощью одного из Python менеджеров: pipили easy_install.

```sh
pip install xlrd
pip install xlwt
```
